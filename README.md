# Library App Catalogue

図書館システム 蔵書カタログサービス

図書館システムにおける蔵書カタログ情報を管理・提供するサービスです。  
書籍シリーズや巻数、タイトルなどの基本情報を保持し、  
フロントエンドからの検索や一覧表示のために REST API を提供します。
